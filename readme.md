# 🗳️ PARTICIPA+

> **Asistente basado en Inteligencia Artificial y Economía Conductual para Presupuestos Participativos.**

---

## 💡 ¿Qué es PARTICIPA+?

**PARTICIPA+** es un asistente virtual diseñado para transformar la participación ciudadana. Combinamos **Inteligencia Artificial** y principios de **Economía Conductual (*Nudge*)** para acompañar activamente a los agentes participativos durante todo el proceso de formulación de propuestas para el presupuesto participativo.

La solución está pensada para:
* 🎯 **Traducir necesidades comunitarias** abstractas o informales en propuestas técnicas estructuradas.
* 🧠 **Reducir las barreras cognitivas** y burocráticas que suelen desmotivar a los ciudadanos.
* 📄 **Generar automáticamente fichas de proyectos** alineadas con los criterios y normativas municipales específicas.

Con esto, logramos una participación mucho más **inclusiva, equitativa y efectiva** en la gestión pública.

---

## 🛠️ Arquitectura y Componentes Abiertos

El proyecto ha sido desarrollado bajo un enfoque de código abierto, promoviendo componentes 100% reutilizables y escalables:

* 🗄️ **Base de Datos:** Estructurada y relacional mediante **PostgreSQL**.
* ⚙️ **Backend:** Servicio robusto y escalable construido con **NodeJS**.
* 💻 **Frontend:** Interfaz de usuario intuitiva y tipada utilizando **TypeScript**.
* 📦 **Control de Versiones & Repositorio:** Hospedado y gestionado de forma colaborativa en **GitHub**.
